# Vehicle-Function-Scenary
一起来创建你使用智能车辆的功能场景，可以抛出一个新功能，所有人可以丰富该功能的使用场景，以及评价标准
我们给每个功能按照3级划分，细分到一个特定功能的时候其索引号为9位，例D01S01F01
D01表示domain 1
S01表示subsystem 1
F01表示function 1
